HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • h1 through h6 determines size of text
  • img allows you to add pictures into a site
  • doctype is not case sensative because it is a declaration
  • the delclaration tells the site which language to expect

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • within color:blue the color is the property and blue is the value
  • box shadow property adds shadow effects around an elements frame

Git

  • git status shows current branch
  • git checkout -b branch-name creates a new branch and switches to it
  • git add -A notes a change to a file
  • git checkout branch-name switches to that already existing branch
